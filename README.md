## Installation

To get started, follow these steps:

1. **Clone the repository**:
    ```bash
    git clone https://github.com/Codevineet/Arc-Tech-Labs-Projects.git
    cd Arc-Tech-Labs-Projects
    ```

2. **Install dependencies**:
    ```bash
    npm install
    ```

3. **Run**:
    ```bash
    npm run dev
    ```
